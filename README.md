# sistema_optimizacion
IntelRR Sistema de Optimización basado en Algoritmos Metaheurísticos Híbridos.
 OBJETIVO GENERAL: Diseñar e implementar un modelo de optimización que combine dos o más algoritmos metaheurísticos (por ejemplo, PSO + Genérico + Búsqueda Tabú) para resolver un problema complejo de asignación, ruteo y planificación, demostrando su eficacia mediante métricas cuantitativas de rendimiento y convergencia.
